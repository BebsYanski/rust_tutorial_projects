# Day 11: Project: Basic Timer Tool

- Build a Command-Line Interface (CLI) Timer Tool in Rust that allows users to set a timer for a specified duration and notifies them when time is up.

## Designing the Timer Tool

- The timer tool will allow the user to:
  - Enter a duration (in hours, minutes, or seconds)
  - Start the timer and display a countdown
  - Notify the user with a "Times's up!" message when the timer ends.
